indexia
=======

``indexia`` is a Python package designed to create, maintain, & display 
hierarchical data models. For source documentation, see :doc:`modules`. For 
definitions of ``indexia`` terms, see the :doc:`glossary`. To learn how 
``indexia`` can be used, see :doc:`usage` & :doc:`applications`.

Installation
------------

``pip install indexia``
